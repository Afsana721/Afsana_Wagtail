

### Create a Calendar by a function expression
1️. Create a function like createCalendar() that receives year & month.
2️. Inside, loop days of month → build 7-column grid (Sun–Sat).
3️. Add click listener on each date → store selected date.
4️. Render month title and next/prev buttons outside the function.
5️. Store tasks in localStorage linked to that date.
6️. On month change → call createCalendar() again with updated month/year.
